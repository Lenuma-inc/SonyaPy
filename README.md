# Голосовой ассистент на python с интеграцией api Chat GPT

## Installation

> Для установки зависимостей и запуска бота c интерфейсом на pyqt используйте эти команды:

```sh
pip install -r requirements-pyqt5.txt
python app-interface.py
```

> Для установки зависимостей и запуска бота без интерфейса используйте эти команды:

```sh
pip install -r requirements.txt
python app.py
```
